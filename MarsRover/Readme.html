Mars Rover Solution

Design Assumptions:

  • All the required inputs are to be passed mandatorily with the format mentioned in the problem statement
  • All inputs are to be given in upper case as mentioned in the problem statement

Project Details:

  • This Solution is developed using C# Language over .Net 4.5 Framework
  • The Solution contains three projects as mentioned below,
    • MarsRover Project containing the actual implementation of the problem statement
    • RoverTests Project containing Test Methods for all possible scenarios in the solution
    • MarsRoverModel Project containing class diagrams and a sequence diagram to illustrate the design

Evaluation Details:

  • There are two approaches this solution can be evaluated as mentioned below
    • In the RoverTests project under MarsRoverControllerTest class following tests are written to confirm the working nature of this solution
      • MarsRoverController_NavigateRover_Pass12N_LMLMLMLMM_Expect13N
      • MarsRoverController_NavigateRover_Pass33E_MMRMMRMRRM_Expect51E
      • MarsRoverController_NavigateRover_Pass55S_LLLMRMRMLLLRRR_Expect55E
      • MarsRoverController_NavigateRover_Pass66S_LRM_ExpectException
      • MarsRoverController_NavigateRover_PassMinus11N_LRM_ExpectException
      • MarsRoverController_NavigateRover_Pass55Error_LRM_ExpectException
      • MarsRoverController_NavigateRover_Pass30W_RMLLMRR_Expect30N
      • MarsRoverController_NavigateRover_Pass00S_MMMMMLMMM_Expect30E
      • MarsRoverController_NavigateRover_Pass50E_MMMMMLMMM_Expect53N
      • MarsRoverController_NavigateRover_Pass50E_MMMMMLMMMMMMRMMMMMLLM_Expect45W
    • By running the application in Console mode and passing the input in the format mentioned in the Problem Statement
